Assessment1

Q1:What’s the quickest way to learn more about each feature in the toolbar?

(用什麼方法能最快的了解在工具欄上的每個特徵?)

A1:Hover the mouse over it for a second, and a description of the feature will pop up.

(將鼠標放在上方一下子，會跳出一個關於此特徵地描述。)

Q2:Briefly describe the four foundational features. Feel free to draw pictures if it helps:

(簡要地描述四個基礎特徵。如果這能有所幫助，可以隨意地畫圖。)

A2:

Extrude: This geometry is created when a constant cross section is extruded in a
                     straight direction

(擠出:以一直線方向擠出一個恆定橫截面的幾何圖形。)

Revolve: This geometry is created when a constant cross-section is revolved
                    around an axis of revolution.

(移除:以一旋轉軸上移除一個恆定橫截面的幾何圖形。)

Sweep: This geometry is created when a constant cross-section is swept along a
                  path.

(掃出:沿著一條路徑掃出一個恆定橫截面的幾何圖形。)

Loft: This geometry is created when multiple cross sections are smoothly
            connected together.

(斷面混成:使多個橫截面平順的相連一起。)

Q3:What is the default geometry in a Part Studio?

(在部分畫室中默認幾何體是什麼?)

A3:The origin and three orthogonal planes: Top, Front, Right.

(起點與三個正交平面:上視圖，前視圖，右視圖。)

Q4:What’s the difference between a box select from Left-to-Right and a box select from
    Right-to-Left? 

(由左至右框選跟從右至左框選差異在哪?)

A4:L-R = selects everything within the box, R-L = selects everything the box touches.

(由左至右 = 框選內部所有內容，從右至左 = 框選所有的接觸內容)

Q5:Can a document in Onshape have multiple Part Studios? 

(一項文件在Onshape上可以開啟多個畫室嗎?)

A5:Yes
